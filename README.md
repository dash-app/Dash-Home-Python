Dash-Home
==============================

## Install
### 1. Install pipenv
```bash
pip install pipenv

# or install as user (linux, osx etc.)
pip install --user pipenv
```

### 2. Install required packages
```bash
pipenv install
```

## Run
```bash
# Run web service on 8081
python ./app.py

# specify port manually (ex. 3000)
HTTP_PORT=3000 ./app.py
```
